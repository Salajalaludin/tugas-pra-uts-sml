Bank Term Deposit Subscription Prediction
Proyek ini bertujuan untuk membangun model klasifikasi machine learning yang mampu memprediksi apakah seorang nasabah akan berlangganan produk deposito berjangka (term deposit) berdasarkan karakteristik demografi, riwayat interaksi, dan indikator ekonomi.

📖 Deskripsi Proyek
Bank ingin meningkatkan efektivitas pemasaran. Masalah utama dalam dataset ini seringkali adalah ketidakseimbangan kelas (class imbalance), di mana jumlah nasabah yang tidak berlangganan jauh lebih banyak daripada yang berlangganan. Fokus utama adalah mengoptimalkan nilai F1-Score untuk menyeimbangkan Precision dan Recall.
